# 🎲 Lucky Number - Sorteador de Números

Um projeto simples e elegante de sorteador de números, desenvolvido para praticar conceitos fundamentais de **JavaScript**, **HTML** e **CSS**.

## 🚀 Demonstração

O usuário pode definir um valor mínimo e máximo, e o sistema gera automaticamente um número aleatório dentro desse intervalo.

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização e responsividade
* **JavaScript** → Lógica do sorteio

## ⚙️ Funcionalidades

* Inserir valor mínimo e máximo
* Validação básica dos inputs
* Geração de número aleatório
* Exibição dinâmica do resultado
* Layout responsivo (mobile e tablet)

## 🧠 Lógica do Sorteio

O número é gerado com base na seguinte fórmula:

```javascript
Math.floor(Math.random() * (max - min + 1)) + min
```
